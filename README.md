# 🎯 Shoot! Shoot  
> Unreal 기반 회사에 지원하기 위해 엔진을 처음 학습하며 제작한 프로젝트입니다. 이 의미가 담긴 개인작을 대체할 키워드 

---

## 📌 개요  
과녁을 조준해 제한 시간 내 목표 점수를 달성하면 승리하는 간단한 1인칭 슈팅 게임입니다.

---

## 🧩 제작한 블루프린트

| 블루프린트 | 역할 |
|------------|------|
| **무기** | 사격 입력 처리 |
| **총알** | 타겟과의 충돌 처리 |
| **게임모드** | 점수 계산, 타이머, 클리어 조건 관리 |
| **UI** | 실시간 점수 및 시간 표시 |
